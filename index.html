XvddKeyslotUtil

Go to project page

A command-line program to identify and extract key material for current installed games through the Xbox desktop app.

Show details about the project

This project was written after reverse engineering the XVDD.sys driver that is included in the GamingServices package for Windows 10. The original goal was to be able to modify Halo: The Master Chief Collection but due to the nature of MSIXVC packages; accessing game data is not possible without decrypting and extracting the data. Therefore, I created a memory dump of a virtual machine and captured the license information belonging to the title. This lead to the discovery of a Key Slot table belonging in the XVDD memory region. Utilising Kernel-Bridge, I was able to seamlessly iterate and locate key data in the kernel memory region through the command line.

Python script to parse an unknown file format belonging to the Xbox platform.

A small script written to parse and extract the individual Windows PE files within the VBI> binary. This file was located in the user-data section of an operating system XVD on the Xbox One console. Further analysis revealed that this binary contained critical boot components: such as the kernel and other important drivers found on Windows.
